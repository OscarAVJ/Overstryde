OVERSTRYDE
Es una aplicación web de comercio electrónico desarrollada con el stack MERN (MongoDB, Express.js, React y Node.js), enfocada en la venta de ropa deportiva para gimnasio. La plataforma ofrece una experiencia de compra moderna, intuitiva y responsiva, permitiendo a los usuarios explorar un catálogo de productos, gestionar su carrito de compras, realizar pedidos y administrar su perfil.

La aplicación cuenta con un sistema de autenticación de usuarios, gestión de productos por categorías, búsqueda y filtrado, administración de pedidos, reseñas de productos y un panel para la gestión del inventario. Toda la información se almacena en MongoDB, mientras que Express.js y Node.js proporcionan una API REST segura y escalable. El frontend está desarrollado con React, ofreciendo una interfaz dinámica y una excelente experiencia de usuario.

- Tecnologías utilizadas
Frontend: React, React Router, Tailwind CSS
Backend: Node.js, Express.js
Base de datos: MongoDB
Autenticación: JWT (JSON Web Tokens)
Control de versiones: Git y GitHub
- Características principales
Registro e inicio de sesión de usuarios.
Catálogo de productos deportivos.
Búsqueda y filtrado por categorías.
Carrito de compras.
Gestión de pedidos.
Reseñas y calificaciones de productos.
Panel de administración para productos y categorías.
Diseño responsive para dispositivos móviles y escritorio.

 Objetivo del proyecto

Desarrollar una plataforma de comercio electrónico completa utilizando el stack MERN, aplicando buenas prácticas de desarrollo web, arquitectura cliente-servidor y gestión de bases de datos NoSQL, ofreciendo una solución moderna para la venta de ropa deportiva destinada a personas que practican actividad física y entrenamiento en gimnasio.
